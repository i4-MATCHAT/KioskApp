# KioskApp

- 상품을 보관하고 사진 촬영을 할 함 제작
- 라즈베라피이로 제어 가능한 카메라 모듈과 서보모터를 함에 설치
- 카메라와 서보모터를 제어하고 함에서 사용자들의 전체적인 거래를 진행하기 위한 키오스크 앱 개발
- 라즈베리파이 내의 로컬 앱으로 tkinter GUI 사용
